# OWASP Juice Shop Setup (Kali Linux)
## 1. Verify Docker

```bash
docker --version
sudo systemctl status docker
sudo systemctl start docker
```

## 2. Pull the image
```bash
sudo docker pull bkimminich/juice-shop
```

## 3. Run the container
```bash
sudo docker run -d -p 3010:3000 --name juice-shop bkimminich/juice-shop
```

## 4. Verify container
```bash
sudo docker ps
```

## 5. Access Juice Shop
```text
http://127.0.0.1:3010
```

## Useful commands
- Stop:
```bash
sudo docker stop juice-shop
```
- Start:
```bash
sudo docker start juice-shop
```
- Remove:
```bash
sudo docker rm -f juice-shop
```
